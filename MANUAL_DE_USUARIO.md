# Manual de Usuario: Sistema de Monitoreo de Aprendizajes

## 1. Acceso al Sistema
Abra la aplicación en su navegador preferido. Al cargar, verá el Dashboard principal con los resultados agregados.

## 2. Carga de Datos (Masiva)
Para cargar notas de una institución educativa completa:
1. Diríjase a la pestaña **"Carga Masiva"** en la barra lateral.
2. Prepare su archivo Excel con las columnas requeridas (DNI, Nombres, IE, Grado, Sección, Periodo y las respuestas de la P1 a la P20).
3. Haga clic en el área de carga o arrastre el archivo.
4. **Validación:** El sistema mostrará una vista previa de los datos detectados.
   - **Modo de Sobrescritura:** Si ya cargó datos anteriormente y desea reemplazarlos (por ejemplo, si hubo un error en los nombres), active este interruptor antes de confirmar.
5. Haga clic en **"Confirmar y Guardar"**.

## 3. Gestión y Borrado de Datos
Si cometió un error al cargar una sección o institución específica:
1. En el **Dashboard**, busque el botón **"Gestionar Datos"** en la esquina superior derecha.
2. Se abrirá un panel con todos los "lotes" de datos cargados (IE, Grado, Sección, Periodo).
3. Utilice la barra de búsqueda para encontrar la institución deseada.
4. Haga clic en el icono de **Papelera** rojo para eliminar ese lote específico.
   - *Nota:* El botón "Limpiar" rápido junto a los filtros solo borra los datos del grado y periodo seleccionados actualmente.

## 4. Visualización de Resultados
### 4.1. Filtros
En la parte superior del Dashboard puede filtrar por **Grado** (1° a 5° de Secundaria) y **Periodo** (Diagnóstica, Inicio, Proceso, Salida).

### 4.2. Pestañas
- **General:** Vista panorámica de los niveles de logro (Satisfactorio, Proceso, Inicio).
- **Competencias:** Desglose por capacidades de lectura/escritura.
- **Alertas Tempranas:** Listado de estudiantes que requieren intervención inmediata (Nivel Inicio).
- **Comparativo:** (Si hay datos de varios periodos) Permite ver el progreso entre evaluaciones.

## 5. Recomendaciones
- Asegúrese de que el **DNI** sea correcto, ya que es el identificador único del estudiante.
- No es necesario cargar todas las preguntas; si un estudiante dejó una en blanco, el sistema la tomará como incorrecta automáticamente.
- El sistema guarda los datos en su navegador. Si limpia el historial/caché de navegación, es posible que los datos se eliminen. Se recomienda realizar copias de seguridad de sus archivos Excel.
